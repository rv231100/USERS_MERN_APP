"Designed a full-stack web application leveraging React for a responsive, component-driven user interface, backed by a Node.js and Express.js RESTful API, and utilizing MongoDB for efficient, schema-based data modeling."

------------------The following points explains the USER_MERN_APP implementation and basic concepts of MERN Stack.----------------------------------------
1. creating class components, props and state, DOM events, promises.
2. React life cycle hooks, fuctional components, useState hook, imitating lifecycle methods.
3. flex-box, responsive web design using Bootstrap.
4. React Routing, reading route and query params using searchQuery and useParams hook.
5. Regex, checking password strength, Bootstrap.
6. REST API, HTTP verbs, creating http server using express, routing.
7. MongoDB, basic,array operators, aggregation pipeline.
8. Mongoose, Schema, validators. Creating end-points for signup, login, add/remove.
9. "Password hashing, Session management using JWT, authMiddleware for protected routes.
10. 2FA- setup via Google Authenticator app, Reset password implementation, env variables.
11. Consuming the api in React application, Intro to CORS, useRef hook, State lifting and prop-drilling.
12. Redux, Async actions using Thunk, useslector and usedispatch hook.
13. Showing visual alerts to users for success as well as error from the backend response.
14. Displaying a loader to indicate api fetching state, React code splitting.
15.  Navigting programatically using useNavigate, useLocation hooks.
16.  Deploying the express server and react app on render
